### 2.2 Telemetry System [BLOCKING]

**First-Run Consent**:
```bash
# Test on fresh install (delete database first)
rm -f ~/.local/share/caro/telemetry.db
caro "list files"
```
- [ ] Consent prompt appears on first use
- [ ] Prompt is clear and understandable
- [ ] User can accept or decline
- [ ] Choice is persisted (no re-prompt on second command)

**Data Collection**:
```bash
# Generate a few commands
caro "list files"
caro "processes"
caro "disk usage"

# Check collected data
caro telemetry show
```
- [ ] Events are recorded
- [ ] Timestamps are correct
- [ ] Platform info is accurate
- [ ] No PII visible in output (manually inspect)

**Export Function**:
```bash
caro telemetry export test-export.json

# Inspect the export
cat test-export.json | jq '.'
```
- [ ] Export creates valid JSON file
- [ ] File contains expected event types
- [ ] No PII in exported data (manually inspect every field)
- [ ] File is readable and well-formatted

**Show Function**:
```bash
caro telemetry show
```
- [ ] Displays events in readable format
- [ ] Shows recent events first
- [ ] Formatting is clear
- [ ] No errors or crashes

**Clear Function**:
```bash
caro telemetry clear
caro telemetry show
```
- [ ] Clears all events
- [ ] Show returns empty or "no events"
- [ ] No errors

**Disable Function**:
```bash
caro config set telemetry.enabled false
caro "list files"
caro telemetry show
```
- [ ] Telemetry respects disabled config
- [ ] No new events collected after disabling
- [ ] Commands still work with telemetry off

## Appendix: Quick Command Reference

### Useful Commands During Pre-Flight

**Check binary platforms**:
```bash
file caro-macos-aarch64    # Should show: Mach-O 64-bit executable arm64
file caro-linux-x86_64     # Should show: ELF 64-bit LSB executable, x86-64
```

**Verify version**:
```bash
./caro --version | grep "1.1.0-beta"
```

**Check crates.io**:
```bash
cargo search caro | grep "1.1.0-beta"
```

**Test telemetry database**:
```bash
sqlite3 ~/.local/share/caro/telemetry.db "SELECT COUNT(*) FROM events;"
```

**Privacy inspection with jq**:
```bash
# Check for common PII patterns
cat export.json | jq -r '.. | strings' | grep -E "@|\.com|/Users/|/home/"
```
