## ğŸ› ï¸ Common Issues & Fixes

### "Installation failed"
- Check platform compatibility
- Verify curl/cargo version
- Test binary install script manually
- Provide direct binary download if needed

### "Command generation too slow"
- Check backend (static vs embedded)
- Ask for generation time estimate
- Check platform resources
- May be expected for complex queries

### "Privacy concerns"
- Ask for specific data point
- Review export with them
- Explain validation layers
- Offer to disable if needed

### "Telemetry export failed"
- Check SQLite file exists: `~/.local/share/caro/telemetry.db`
- Verify permissions
- Try: `caro telemetry show` first
- Manual export path if needed

## ğŸ”„ If Things Go Wrong

### Scenario: Critical Bug on Jan 15
1. Ask tester to stop testing
2. Assess severity (P0 vs P1)
3. Hot fix within 4 hours if P0
4. Extend beta by 1 day if needed
5. Re-test with same tester
6. Document in incident log

### Scenario: Low Survey Response on Jan 17
1. Personal follow-up emails
2. Offer call/chat instead of survey
3. Accept partial responses
4. Don't delay go/no-go unnecessarily

### Scenario: Privacy Leak Found
1. **STOP EVERYTHING IMMEDIATELY**
2. Use incident response plan (P0)
3. Notify all testers within 1 hour
4. Fix validation layer (emergency)
5. Re-test all exports manually
6. No release until verified safe
