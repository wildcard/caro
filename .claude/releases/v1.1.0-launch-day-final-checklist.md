## Rollback Procedure (if needed)

**Criteria for Rollback**:
- P0 bug affecting > 50% of users
- Security vulnerability discovered
- Data loss or corruption
- Installation failure > 80%

**Rollback Steps**:
1. [ ] Announce rollback on Discord/social media
2. [ ] Un-publish GitHub release (mark as draft)
3. [ ] Yank crates.io version (`cargo yank --vers 1.1.0-beta`)
4. [ ] Revert website to previous version
5. [ ] Post-mortem: What happened? Why? How to prevent?

**Rollback Decision Maker**: Release Manager + Engineering Lead
