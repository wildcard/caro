---

### "A P0 bug was just reported"
1. **STOP** - don't panic
2. Open: [Incident Response Plan](v1.1.0-beta-incident-response-plan.md)
3. Follow: 6-phase incident workflow
4. Create: GitHub issue with P0 label
5. Notify: Team via defined channels
6. Mitigate: Within time SLA (see plan)
7. Document: Every step for post-mortem

---

### "We found PII in telemetry"
1. **EMERGENCY STOP** - highest priority
2. Open: [Incident Response Plan](v1.1.0-beta-incident-response-plan.md) ¬ß Privacy Incident
3. Disable: ALL telemetry immediately
4. Notify: All users within 15 minutes
5. Investigate: Validation layer failure
6. Fix: Before any further releases
7. Result: BLOCKS release until verified safe
