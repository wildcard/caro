- **Rust**: Async (tokio), fast (<500ms p95), single binary
- **3 backends**: Static matcher (patterns), embedded ML (local), Ollama (optional)
- **Safety validation**: Blocks rm -rf, dd, privilege escalation, destructive ops

Every command goes through validation:
```rust
pub struct SafetyValidator {
    rules: Vec<SafetyRule>,
}

pub enum Severity {

    Warn,     // Show warning (sudo, chmod 777)

}
```

YOU approve before execution. Always.
