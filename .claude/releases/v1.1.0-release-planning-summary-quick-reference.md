## Quick Command Reference

### Testing
```bash
# Run all tests
cargo test --all-features

# Run specific test suite
cargo test --test integration_tests

# Run beta test case
caro test --suite .claude/beta-testing/test-cases.yaml --profile bt_001
```

### Release
```bash
# Tag release
git tag -a v1.1.0-beta -m "Release v1.1.0-beta"
git push origin v1.1.0-beta

# Build release binaries (via CI/CD)
# Triggered automatically on tag push

# Verify installation
caro --version
```

### Monitoring
```bash
# Check GitHub release stats
gh release view v1.1.0-beta

# Monitor Discord activity
# Manual: Check #announcements, #support, #bugs

# Check metrics (if telemetry enabled)
caro stats
```

### Rollback
```bash
# Un-latest v1.1.0-beta
gh release edit v1.1.0-beta --draft

# Mark v1.0.0 as latest
gh release edit v1.0.0 --latest

# Update install script
sed -i 's/VERSION=v1.1.0-beta/VERSION=v1.0.0/' install.sh
git commit -am "chore: rollback to v1.0.0"
git push
```
