### Appendix C: Rollback Procedure

**If rollback required**:

1. **Yank from crates.io**:
   ```bash
   cargo yank --vers 1.1.0-beta
   ```

2. **Delete GitHub Release** (optional):
   ```bash
   gh release delete v1.1.0-beta
   ```

3. **Revert Homebrew Tap**:
   ```bash
   cd homebrew-tap
   git revert HEAD
   git push
   ```

4. **Communicate**:
   - GitHub: Update release notes with "YANKED" notice
   - Discord: Announce rollback
   - Twitter: Tweet status
   - Email: Notify users (if email list exists)

5. **Fix and Re-release**:
   - Address critical issue
   - Release v1.1.1 with fix
