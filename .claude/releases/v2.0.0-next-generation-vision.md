#### F2: Advanced Context Engine

**Value**: Deep understanding of your development environment

**Context Sources**:

1. **Git Repository**
```bash
# Caro understands your repo
caro "show files changed in this feature"
# Knows current branch, reads git log
# Generated: git diff main..HEAD --name-only

# Generated: git log --follow -- src/auth/mod.rs

caro "undo changes to config file"
# Knows if file is staged, committed, or unstaged
# Generated: git checkout -- config.toml
```

2. **Docker Containers**
```bash
# Caro introspects running containers
caro "show logs from web container"
# Reads docker ps, finds container named "web"
# Generated: docker logs web-container-abc123

caro "exec into the database"
# Finds running postgres container
# Generated: docker exec -it postgres-container /bin/bash
```

3. **Kubernetes Cluster**
```bash
# Caro reads k8s context
caro "scale api deployment to 10"
# Validates deployment exists in current namespace
# Generated: kubectl scale deployment/api --replicas=10

caro "show failed pods"
# Generated: kubectl get pods --field-selector=status.phase=Failed
```

4. **CI/CD Pipeline**
```bash
# Caro integrates with GitHub Actions
caro "rerun failed tests"
# Identifies failed workflow, triggers rerun
# Generated: gh workflow run tests.yml

caro "show last 5 deployment logs"
# Generated: gh run list --workflow=deploy.yml --limit=5
```

**Context Engine Architecture**:

```rust
pub struct ContextEngine {
    git: GitContext,
    docker: DockerContext,
    k8s: KubernetesContext,
    cicd: CICDContext,
}

pub struct GitContext {
    pub repo_root: PathBuf,
    pub current_branch: String,
    pub remote: String,
    pub status: GitStatus,
}

impl ContextEngine {
    pub async fn enrich_prompt(&self, prompt: &str) -> String {
        let mut enriched = prompt.to_string();

        // Add git context
        if self.git.in_repo() {
            enriched.push_str(&format!(
                " [git: branch={}, status={}]",
                self.git.current_branch,
                self.git.status
            ));
        }

        // Add docker context
        if self.docker.containers_running() {
            enriched.push_str(&format!(
                " [docker: {} containers]",
                self.docker.count()
            ));
        }

        enriched
    }
}
```

**Privacy Considerations**:
- Context read-only (never modifies)
- Sensitive data filtered (tokens, keys)
- User controls which context sources enabled
- Context never sent to cloud (local only)
