# Contributors

This file acknowledges everyone who has contributed to cmdai. Every contribution, no matter how small, helps make the terminal safer and more accessible for everyone.

## How to Get Listed

You'll be added to this file when:
- Your pull request is merged
- You submit a safety pattern that's incorporated
- You help with documentation, testing, or issue triage
- You contribute in any meaningful way to the project

**Note:** This file is automatically updated by maintainers. If you've contributed but don't see your name, please open an issue!

## Special Recognition

### First Contributions

We celebrate first-time contributors! Your first merged PR earns you:
- A welcome shoutout in the PR
- Listing in the next release notes

- Our eternal gratitude

### Milestone Achievements

We recognize significant contribution milestones:

**10+ Contributions:**
- Invitation to be a project reviewer
- Recognition in release announcements
- Opportunity for technical blog posts

**25+ Contributions:**
- Invitation to project maintainer team

- Authority in technical decisions

**50+ Contributions:**
- Project core team member

## How We Recognize Contributors

### In Pull Requests

When your PR is merged:
- Maintainer adds a thank you comment
- PR is labeled with contribution type
- You're mentioned in the merge commit

### In Release Notes

Each release acknowledges contributors:
```markdown
## [0.3.0] - 2025-11-30

### Added
- Feature X (#123) - @your-username
- Safety pattern Y (#124) - @your-username

### Fixed
- Bug Z (#125) - @your-username

Special thanks to @contributor1, @contributor2, and @contributor3 for making this release possible!
```

### In CHANGELOG.md

All contributions are documented with attribution:
- Feature additions credit the implementer
- Bug fixes credit the discoverer and fixer
- Documentation improvements credit the writer
- Performance improvements show before/after metrics

### On Social Media (Future)

We plan to celebrate contributions via:
- Project Twitter/Mastodon account
- Blog posts highlighting major features
- Conference talks mentioning contributors
- Community newsletters
