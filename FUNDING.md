# Funding

Caro is open source software maintained by [@wildcard](https://github.com/wildcard). Your support helps keep the project sustainable and enables continued development.

## Ways to Support

### GitHub Sponsors (Recommended)

The easiest way to support Caro is through GitHub Sponsors:

[![Sponsor wildcard](https://img.shields.io/badge/Sponsor-%E2%9D%A4-ea4aaa?style=for-the-badge&logo=github-sponsors)](https://github.com/sponsors/wildcard)

**[Sponsor on GitHub](https://github.com/sponsors/wildcard)**

GitHub Sponsors offers:
- One-time or recurring donations
- No platform fees (GitHub covers them)
- Direct support to the maintainer

### Open Collective

For organizations preferring transparent, accountable funding:

**[Support on Open Collective](https://opencollective.com/carosh)**

Open Collective offers:
- Full transparency on how funds are used
- Invoice/receipt generation for businesses
- Expense tracking and reporting

### Ko-fi

For one-time contributions:

**[Buy me a coffee on Ko-fi](https://ko-fi.com/kobikadosh)**

Ko-fi offers:
- Simple one-time donations
- No platform fees on donations
- Quick and easy checkout

## What Your Support Enables

- Continued maintenance and bug fixes
- New feature development
- Security updates and audits
- Documentation improvements
- Community support

## Other Ways to Contribute

Not able to contribute financially? You can still help:

- **Star the repo** - helps with visibility
- **Report bugs** - [open an issue](https://github.com/wildcard/caro/issues)
- **Contribute code** - [see contributing guide](https://github.com/wildcard/caro/contribute)
- **Spread the word** - tell others about Caro
