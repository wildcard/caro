## Hashtag Research & Tracking

### How to Evaluate Hashtag Performance

**For each hashtag, track:**
1. **Volume:** How many posts use it? (too high = lost in noise)
2. **Engagement:** Average engagement on posts with this tag
3. **Relevance:** Is the audience our target users?
4. **Competition:** Are we competing with major brands?
5. **Growth:** Is the hashtag growing or declining?

### Tools to Use
- **Twitter Analytics** - Built-in performance data
- **Hashtagify** - Hashtag correlation and trending
- **RiteTag** - Real-time hashtag suggestions
- **Sprout Social** - Cross-platform hashtag analytics

### Monthly Review Process
1. Export post performance data
2. Correlate with hashtag usage
3. Identify top-performing hashtags
4. Retire low-performing ones
5. Test 2-3 new hashtags
6. Update this document

## Hashtag A/B Testing

### Month 1 Tests

**Test 1: Security vs. Safety**
- Group A: #TerminalSafety
- Group B: #TerminalSecurity
- Measure: Engagement rate, CTR
- Hypothesis: "Safety" resonates more (less intimidating)

**Test 2: Rust Branding**
- Group A: #RustLang
- Group B: #MadeWithRust
- Measure: Engagement from Rust community
- Hypothesis: #RustLang has more reach

**Test 3: AI Philosophy**
- Group A: #ResponsibleAI
- Group B: #SafeAI
- Measure: Engagement rate
- Hypothesis: #SafeAI more action-oriented

### How to Test
1. Post similar content with different hashtag sets
2. Control for posting time/day
3. Track engagement for 7 days
4. Statistical significance check
5. Implement winner

## Crisis Management

### If a Hashtag Gets Hijacked

**Scenario:** Someone uses #cmdai for spam or controversial content

**Response:**
1. Don't engage with hijacker
2. Continue using hashtag normally
3. Report spam/abuse to platform
4. Flood with quality content
5. If persistent, consider new hashtag

### If a Hashtag Goes Viral (For Wrong Reasons)

**Scenario:** #SafeAI becomes associated with AI safety debates (not our focus)

**Response:**
1. Continue using with clear context
2. Add clarifying hashtags (#TerminalSafety, #DevTools)
3. Don't try to "own" broad hashtag
4. Focus on #cmdai as primary
