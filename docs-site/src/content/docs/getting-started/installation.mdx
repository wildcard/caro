## macOS

caro is available via Homebrew, Cargo, and as a standalone binary.

### Homebrew

| Install | Upgrade |
| ------- | ------- |
| `brew install wildcard/tap/caro` | `brew upgrade caro` |

### Cargo

```bash
cargo install caro

# For Apple Silicon with MLX GPU acceleration:
cargo install caro --features embedded-mlx
```

### Precompiled binaries

Download from the [releases page](https://github.com/wildcard/caro/releases/latest) or use the install script:

```bash
curl -fsSL https://setup.caro.sh | bash
```

<Aside type="tip" title="Apple Silicon">
  For maximum performance with MLX GPU acceleration on M1/M2/M3/M4 Macs, use `cargo install caro --features embedded-mlx` or install via Homebrew.
</Aside>

## Build from source

```bash
git clone https://github.com/wildcard/caro.git
cd caro
cargo build --release
./target/release/caro --version
```

**Prerequisites**: Rust 1.83+, CMake. For Apple Silicon GPU acceleration, install Xcode.

## Verifying Installation

After installation, verify everything is working:

```bash
# Check version
caro --version
# Output: caro 0.1.0

# Test basic functionality
caro "show current directory"
# Output: pwd

# Test with a more complex query
caro "find all markdown files"
# Output: find . -name "*.md"
```

## First Run

On first run, caro will download the required model (~1.1GB):

```bash
$ caro "list files"
Downloading model... [====================] 100%
Model cached at ~/.cache/caro/models/

find . -type f
```

<Aside type="note">
  The model is cached locally and won't be downloaded again unless you clear the cache or update to a new version.
</Aside>

## Configuration

caro stores configuration in:
- **macOS**: `~/Library/Application Support/caro/`
- **Linux**: `~/.config/caro/`
- **Windows**: `%APPDATA%\caro\`

Model cache location:
- **macOS**: `~/Library/Caches/caro/models/`
- **Linux**: `~/.cache/caro/models/`
- **Windows**: `%LOCALAPPDATA%\caro\cache\`

## Troubleshooting

### "command not found: caro"

Ensure Cargo's bin directory is in your PATH:

```bash
# Add to ~/.bashrc or ~/.zshrc
export PATH="$HOME/.cargo/bin:$PATH"

# Reload shell
source ~/.bashrc  # or ~/.zshrc
```

### Build fails with CMake error

Install CMake:

```bash
# macOS
brew install cmake

# Ubuntu/Debian
sudo apt install cmake

# Fedora
sudo dnf install cmake
```

### Model download fails

Check your internet connection and try again:

```bash
# Clear partial download
rm -rf ~/.cache/caro/models/

# Retry
caro "test"
```
