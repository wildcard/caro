## Keyword Research Process

### For New Topics

1. Start with seed keyword from cluster
2. Use tools (Ahrefs, SEMrush, Google) to find related terms
3. Analyze search intent for each term
4. Check competitor content and gaps
5. Prioritize based on volume + difficulty + business value
6. Add to this document and content calendar

### Monthly Review

- Check ranking changes for tracked keywords
- Identify new keyword opportunities
- Update priority scores based on performance
- Add new long-tail discoveries
