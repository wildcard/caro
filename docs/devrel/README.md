# Developer Relations (DevRel) Documentation

This directory contains all developer relations and community engagement materials for the Caro project.

## Current State (January 2025)

**Project:** Caro v1.0.3
**Website:** https://caro.sh
**Tagline:** "Your loyal shell companion"
**Repository:** https://github.com/wildcard/caro
**License:** AGPL-3.0

**Brand Story:** Kyaro (beloved Shiba Inu) → Caro, inspired by Portal's Caroline → GLaDOS transformation - loyalty transformed into eternal companionship.

**Key Offerings:**
- MCP server for Claude Desktop
- Standalone CLI tool (Rust, cross-platform)
- Claude Code Skill integration
- POSIX shell command generation with safety-first design
