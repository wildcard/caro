## Philosophy

These legal documents are written in the spirit of:

- **Transparency** - Clear language, not legalese
- **Minimalism** - Only what's necessary
- **User Respect** - Your data is yours
- **FOSS Values** - Inspired by GNU, BSD, and the open source tradition
