## 10. Key Insights for Reaching These Audiences

### Where They Discover New Tools

1. **Hacker News** - Frontpage exposure drives massive adoption
2. **Reddit** - r/commandline, r/rust, r/linux for organic discovery
3. **GitHub Trending** - Daily monitoring by power users
4. **Terminal Trove** - Curated discovery for CLI enthusiasts
5. **Console.dev** - Trusted by CTOs and senior engineers
6. **Product Hunt** - For launch momentum (CLI tools category)
7. **Lobsters** - High-quality technical discussion

### Preferred Content Formats

1. **Technical blog posts** with benchmarks and comparisons
2. **Video demos** showing terminal workflows
3. **Podcasts** - Long-form technical discussions
4. **Zines** - Julia Evans style, visual learning
5. **GitHub READMEs** - Comprehensive documentation
6. **asciinema recordings** - Terminal session sharing

### Community Values

- **Speed and performance** - Benchmarks matter
- **Composability** - Unix philosophy, pipes and text
- **Open source** - Strong preference for FOSS
- **Documentation** - Man pages, examples, tutorials
- **Minimal dependencies** - Single binary preference
- **Cross-platform** - Linux + macOS + Windows support

### Best Days/Times

- **Hacker News** - Weekday mornings (US time) for launches
- **Product Hunt** - 12:01 AM PST launch optimal
- **Reddit** - Weekday evenings, weekend mornings
- **Newsletters** - Monday (weekly) or Tuesday delivery
