# Presentation Assets

Place your project assets in this directory:

## Required Assets

### mascot.gif
- Your project mascot GIF animation
- Will appear on slide 2 (image-right layout)
- Recommended size: 400x400px or similar
- Format: Animated GIF

## Optional Assets

You can add additional images, logos, or media files here:

- `logo.png` - Project logo
- `demo-screenshot.png` - Demo screenshots
- `architecture.png` - Custom architecture diagrams
- `team-photos/` - Team member photos

## Usage in Slides

Reference assets using relative paths:

```markdown
![Alt text](/asset-name.png)

# Or as background
---
background: /asset-name.jpg
---
```

## Image Guidelines

- **Screenshots**: 1920x1080 or 16:9 aspect ratio
- **Icons**: SVG preferred, PNG as fallback
- **Photos**: Optimized for web (< 500KB)
- **GIFs**: Keep file size reasonable (< 2MB)
