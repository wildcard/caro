macOS (Apple Silicon) macOS (Intel) Linux (x86_64)

Quick Start

1. Install Ollama or configure your preferred backend

2. Run cmdai with a natural language description:

cmdai "list all files larger than 100MB"

3. Review the generated command and confirm to execute
