# Public Assets

This folder contains static assets for the Caro website.

## Caro Pixel Art

Please add the Caro 8-bit pixel art character image to this folder:

**File:** `caro-pixel.png`

The main pixel art character should be the larger version (160x160px) showing Caro sitting.

You can find the pixel art images in the project attachments. Save the main character as `caro-pixel.png` in this directory.

### Image Specifications

- **Format**: PNG with transparency
- **Recommended size**: 160x160px or larger
- **Usage**: Displayed in the hero section with pixelated rendering
- **Animation**: Gentle floating animation applied via CSS

### Other Variations

You can also add the smaller icon versions for use elsewhere:
- `caro-icon-32.png` - 32x32px icon
- `caro-icon-48.png` - 48x48px icon
- `caro-icon-64.png` - 64x64px icon

## Current Files

- `favicon.svg` - Site favicon with Caro gradient and dog emoji
- `caro-pixel.png` - **(Add this)** Main 8-bit Caro character

## Adding Images

Simply drop the PNG files into this folder and they'll be available at `/filename.png` in the built site.
